# Port-Scanner-using-python
Developed a multithreaded GUI-based TCP port scanner in Python with OS detection, banner grabbing, logging, and report export functionality.
Port Scanner

A multithreaded Python-based port scanner with a Kali Linux dark-themed GUI.

## Features

- Multithreaded TCP Port Scanning
- Banner Grabbing
- TTL-Based OS Detection
- Progress Bar
- Save Reports (TXT, JSON, CSV)
- Logging System
- Dark Kali Linux Style Interface

## Built With

- Python 3
- Tkinter
- Socket Programming
- Threading
